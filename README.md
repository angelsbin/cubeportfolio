Cube portfolio jQuery plugin.

[View full demo.](http://scriptpie.com/cubeportfolio/live-preview/)


## Installation

You can install the package via npm or yarn package managers:

```bash
# NPM
npm install cubeportfolio

# Yarn
yarn add cubeportfolio
```
